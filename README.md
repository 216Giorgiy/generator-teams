## How to...

### Install generator

```
npm install generator-teams-tab --global
```

### Use the generator

```
yo teams-tab
```

### Build the generator

```
npm install 
webpack
```

### Use it in dev mode

In the generator-teams tab directory use the following command

```
npm link
```
